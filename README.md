# matthieu-joulot.github.io

Personal academic website of Matthieu Joulot, PhD student in neuroimaging at the Paris Brain Institute (Aramis Lab).

Built with the [Academic Pages](https://github.com/academicpages/academicpages.github.io) Jekyll template.

## Run locally

```bash
bundle exec jekyll serve --host localhost
```

The site will be available at `http://localhost:4000`.
